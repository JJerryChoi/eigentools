## Tools

| Tool | Description |
|------|-------------|
| **JSON Formatter** | Format, minify, and validate JSON |
| **Base64** | Encode and decode Base64 strings |
| **JWT Decoder** | Decode and inspect JSON Web Tokens |
| **URL Encoder** | Encode and decode URLs |
| **UUID Generator** | Generate random v4 UUIDs |
| **Hash Generator** | Generate SHA-1, SHA-256, SHA-512 hashes |
| **Regex Tester** | Test regular expressions with live matching |
| **Timestamp Converter** | Convert between Unix timestamps and dates |
| **Color Converter** | Convert between HEX, RGB, and HSL colors |
| **Markdown Preview** | Live markdown-to-HTML preview with GFM support |
| **Diff Checker** | Compare two texts and highlight differences |
| **YAML / JSON** | Convert between YAML and JSON formats |
| **CSV to JSON** | Convert CSV data to JSON array |
| **Cron Parser** | Parse cron expressions with next run times |

## Why EigenTools?

- **Private** — Everything runs client-side. Zero network requests for your data.
- **Fast** — Static site with no backend. Loads instantly.
- **Free** — No accounts, no limits, no ads.
- **Keyboard-first** — Cmd+K command palette to jump between tools.

## Tech Stack

- [Next.js 16](https://nextjs.org/) (App Router, static export)
- [React 19](https://react.dev/)
- [TypeScript 5](https://www.typescriptlang.org/)
- [Tailwind CSS 4](https://tailwindcss.com/)
- Deployed on [Vercel](https://vercel.com/)

## Getting Started

```bash
git clone https://github.com/JJerryChoi/eigentools.git
cd eigentools
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see it running.
